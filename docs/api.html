Totem Player API

Controla la reproducción en tu Android TV remotamente.

Información General

El servidor corre localmente en el dispositivo Android TV.

  • Puerto por defecto: 8085
  • Prefijo base: /v1
  • Base URL: http://<TV_IP>:8085/v1

1. Reproducir Contenido

Inicia la reproducción de un vídeo. Si hay algo sonando, lo interrumpe inmediatamente.

POST /play

Body (JSON):

{ "url": "https://ejemplo.com/video.mp4", // Requerido "title": "Big Buck Bunny", // Opcional "posterUrl": "https://...", // Opcional (Fondo) "userAgent": "TotemClient/1.0", // Opcional "metadata": "Extra info" // Opcional "startPosition": "0" // Opcional }

2. Control Remoto

Envía comandos de transporte y configuración al reproductor activo.

POST /command

Body (JSON):

{ "command": "PLAY", "value": "" // Opcional según comando }

Comandos Soportados:

Comando Valor (value) Descripción
PLAY-Reanuda la reproducción.
PAUSE-Pausa la reproducción.
STOP-Detiene y vuelve al inicio.
SEEKLong (ms)Salta a la posición (ej: "15000").
FORWARD-Avanza 10 segundos.
REWIND-Retrocede 10 segundos.
METADATAStringCambiar el subtitulo de la imagen
MESSAGEStringEnviar mensaje a la pantalla
VOLUMEFloat (0.0 - 1.0)Fija volumen (ej: "0.5").
SPEEDFloatFija velocidad (ej: "1.5").
POSTERStringCambia la url de la imagen principal
VIDEOString (iso)Cambia audio (ej: "es", "en").
AUDIOString (iso)Cambia audio (ej: "es", "en").
SUBTITLEString (iso)Cambia subtítulo (ej: "en", "off").

3. Obtener Estado

Realiza polling para sincronizar la interfaz de tu mando a distancia.

GET /status

Respuesta Exitosa (200 OK):

{ "state": "PLAYING", // IDLE, BUFFERING, PLAYING, PAUSED... "currentPosition": 15000,// ms actuales "duration": 60000, // ms totales "isLive": false, "volume": 1.0, "speed": 1.0, "title": "Big Buck Bunny", "audioTracks": List_TrackDefinition, "subtitleTracks": List_TrackDefinition, "videoQualities": List_TrackDefinition }

Error 503: Si el reproductor no está inicializado.
